📋 What You Need to Do:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Create Firebase Project (2 minutes)
   → Go to: https://console.firebase.google.com/
   → Click "Add project"
   → Name: "second-brain"
   → Disable Analytics (optional)
   → Click "Create project"

STEP 2: Enable Services (2 minutes)
   → Enable Authentication (Email/Password)
   → Enable Firestore Database (Production mode)
   → Set Firestore Security Rules (see README_FIREBASE.md)

STEP 3: Get Your Config (1 minute)
   → Project Settings → Your apps → Add web app
   → Copy the firebaseConfig object

STEP 4: Add Config to App (1 minute)
   → Open: second-brain/lib/firebase.ts
   → Replace placeholder values with your actual config
   
   OR
   
   → Copy .env.local.example to .env.local
   → Add your Firebase values

STEP 5: Test It! (1 minute)
   → Restart dev server: npm run dev
   → Create account at http://localhost:3000
   → Add some data
   → Open on phone (http://YOUR_IP:3000)
   → Login with same account
   → See data sync! 🎉

🎯 The Problem You Had:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   → Phone has its own data
   → Laptop has its own data
   → No sync between devices ❌

AFTER: Data stored in Firebase Firestore
   → All devices share same data
   → Real-time sync across devices
   → Login from anywhere ✅
